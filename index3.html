Déploiement d’un site Web

1. Qu’est-ce que le déploiement ?

Le déploiement (ou publication) est le processus qui consiste à rendre ton site web accessible au public sur Internet. Avant le déploiement, ton site n’existe que sur ton ordinateur (en local). Après déploiement, il devient visible via une adresse web (nom de domaine).

2. Étapes pour déployer un site web

Choisir l’hébergement : L’hébergement, c’est “l’endroit” où ton site est stocké. Il y’a trois types d’hébergement.

Gratuit

Utilisez GitHub Pages, Netlify ou Vercel pour héberger gratuitement votre site.

Payant

Optez pour un hébergement professionnel avec nom de domaine personnalisé.

Serveur personnel

Hébergez votre site sur un serveur que vous gérez vous-même pour un contrôle total.

Choisir un nom de domaine : Le nom de domaine est l’adresse web de votre site. Il doit être unique et représentatif de votre contenu.C’est l’adresse de ton site, ex. : www.tonsite.com. Tu peux l’acheter via un registrar (OVH, Namecheap, Hostinger…).

Mettre en ligne : Tu as deux grands choix :

Option 1 : Hébergement payant

Simple et rapide : vous payez un abonnement et obtenez un domaine + un espace en ligne.

Option 2 : Serveur personnel

Pour les développeurs avancés : configurez votre propre serveur et DNS.
